# 🛠️ codebrib.dev - Admin Dashboard

Mein persönliches Dashboard für die tägliche Arbeit in der Systemintegration (Windows/Linux).

## 🚀 Features
- **Zentrale Linksammlung:** Schneller Zugriff auf M365, Azure, vCenter.
- **Download-Toolbox:** Direkte Links zu essentieller Software (Wireshark, Putty, etc.).
- **CLI Snippets:** Copy-Paste Vorlagen für PowerShell und Bash.
- **Local Storage Scratchpad:** Notizen bleiben im Browser gespeichert.

## 💻 Lokale Entwicklung
Einfach die `index.html` in einem beliebigen Browser öffnen. Keine Datenbank oder Webserver erforderlich.

## 🌐 Deployment
Wird automatisch via **GitHub Pages** unter `https://[banomx].github.io/codebrib.dev/` veröffentlicht.
